CMS2CMS Connector

== Description ==
CMS2CMS Connection Plugin is designed to make your website migration become an effortless, flawless and automated process. This plugin establishes the connection between your Existing website and your New WordPress site.

= DOCS & SUPPORT =
To find more detailed information about CMS2CMS Connector Plugin, you can check FAQ. If you were unable to find the answer to your question on the FAQ or in any of the documentation, you should check the CMS2CMS support forum on WordPress.org. If you can’t locate any topics that pertain to your particular issue, post a new topic for it.

= Major Advantages of Migration with CMS2CMS: =
* Free Demo Migration helps see the migration outcome beforehand;
* The average migration time is from several minutes till several hours. Timing depends on how many content items you’re going to move from your Existing website to WordPress;
* Migrate the content you want. Choose how you want it to be and look on WordPress  (Data ‘Mapping’ option);
* Complete data migration price depends on the content volume you’re about to migrate to WordPress.

= Compatible with: WordPress 4.1.x - 4.8.x. =
- Live help is available. Our support team is here 24/7.
- When a custom migration is needed, CMS2CMS offers data migration service packages to order.

== Installation ==

1. Download the plugin zip file
1. Extract plugin zip file to your PC
1. Upload extracted file to wp-content/plugin directory
1. Go to Admin -> Plugins, find “CMS2CMS Connector” and click Activate
1. Register an account
1. Setup Demo Migration
1. You’ll be redirected to CMS2CMS website in order to complete your migration
